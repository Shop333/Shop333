Saya adalah seorang **Junior Full Stack Developer** yang berbasis di Jembrana, berfokus pada pembuatan aplikasi web berperforma tinggi dan antarmuka mobile yang modern. Saya senang menggabungkan fungsionalitas dengan desain *pixel-perfect*.

- 📱 **Mobile Dev:** Mengembangkan aplikasi menggunakan **Kotlin** dan **AndroidIDE**.
- 🛠️ **Workflow:** Produktif menggunakan **Termux** dan **Acode** di lingkungan mobile.
- ⚡ **Minat:** Terobsesi dengan *UI/UX Glassmorphism* dan optimasi performa web.
